## 探索順位

* メソッドと変数の探索順位は変数が先です。

```ruby
>> hoge = 0
=> 0
>> def hoge
>>   x = 0
>>   5.times do |i|
?>     x += 1
>>   end
>>   x
>> end
=> :hoge
>> puts hoge
0
=> nil
```
