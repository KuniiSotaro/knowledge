07 if文と組み込み型
=================

## 組み込み型とTrue、False

* Pythonでは、長さゼロの文字列やシーケンスをbool型として評価した時に、False(偽)としてみなす

  => 逆に、長さがゼロでないシーケンスはTrue(真)としてみなす

* 例)文字列sがから出ないことを調べる

```python
# 通常の書き方
if len(s) != 0:

# 簡潔な書き方
if s:
```



### True

Pythonでは、以下のような組み込み型のオブジェクトをbool値として評価した時に、True(真)を返す

* 0以外の数値

* 長さのある文字列(空出ない文字列)

* 要素を持つ、リストやタプルのようなシーケンス

* 要素を持つディクショナリ



### False

* 数値の0

* 空の文字列(空の文字列は「""」として定義できる)

* 空のリストやタプル(空のリストは[]として定義できる)

* 空のディクショナリ(空のディクショナリは{}として定義できる)
